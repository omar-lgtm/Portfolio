# Yacht Game

Python implementation of the Yacht dice game (similar to Yahtzee).

- Players roll 5 dice per round and choose scoring categories (Ones to Sixes).  
- Tracks total score and updates a high scores file (`High_Scores.txt`).  
- User-friendly terminal interface.

## How to Run

1. Make sure Python is installed on your computer.
2. Run `python yacht_game.py` in the terminal.
3. Follow the prompts to play the game.

## Files

- `yacht_game.py` – main game code  
- `High_Scores.txt` – stores top scores
